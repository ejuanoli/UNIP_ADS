Para logar, utilize:
username: admin

Como a criação de usuários exige uma aprovação por parte da administração, é necessário primeiramente existir o usuário admin.

Dentre os arquivos no repositório, temos o manual do usuário e os slide explicaticos, assim como o código para facilitação no uso.

Sobre o código, para que a conexão local funcione em mais de uma máquina, é necessário configurar a porta dentro do servidor e é necessário estar na mesma rede local, sendo necessário colocar o Host da máquina em que a porta foi configurada.
